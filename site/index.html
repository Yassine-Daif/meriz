Modéliser en Merise,
simplement, partout.

Meriz est l'alternative moderne à AnalyseSI. Dessinez votre MCD, l'outil vérifie votre modèle puis génère le MLD, le MPD et le script SQL. Dans le navigateur ou en logiciel, libre et gratuit.

Ce que Meriz fait pour vous

  • Éditeur visuel de MCD

    Entités, associations, cardinalités : tout se dessine à la souris ou au clavier, avec un dictionnaire central des propriétés.

  • Vérification en direct

    Erreurs bloquantes et avertissements s'affichent pendant que vous travaillez, chacun cliquable vers l'élément concerné.

  • Du MCD au SQL

    Le MLD, le MPD et le script CREATE TABLE sont générés selon les règles Merise, en MySQL ou PostgreSQL.

  • Pensé pour apprendre

    Une page Apprendre explique la méthode, les cardinalités et les règles de passage, avec le même exemple du début à la fin.

Télécharger le logiciel

La même application que la version web, installée sur votre machine, qui marche hors ligne.

Windows (.exe) macOS (.dmg) Linux (.deb, .AppImage)

Téléchargement direct, avec le fichier conseillé pour votre système. Ou utilisez la version web, sans rien installer.
