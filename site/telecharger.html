Recommandé pour votre système

Télécharger

Tous les téléchargements

Windows

  • Installateur (.exe) conseillé
  • Paquet MSI (.msi)

À savoir : au premier lancement, Windows affichera « Windows a protégé votre ordinateur ». C'est normal : le logiciel est open source et non signé, il n'y a rien de dangereux. Cliquez « Informations complémentaires » puis « Exécuter quand même ».

macOS

  • Mac Apple Silicon (.dmg) M1 et plus récents
  • Mac Intel (.dmg)

À savoir : au premier lancement, macOS bloquera l'ouverture (application non signée, rien de dangereux). Faites un clic droit sur Meriz puis « Ouvrir » : une seule fois suffit.

Linux

  • Debian, Ubuntu (.deb) conseillé
  • Toutes distributions (.AppImage)
  • Fedora, openSUSE (.rpm)

À savoir : pour l'AppImage, rendez le fichier exécutable avant de le lancer (clic droit, Propriétés, ou chmod +x). Les paquets .deb et .rpm s'installent sans étape particulière.

Pas envie d'installer ? La version web fait la même chose, directement dans le navigateur. Sommes de contrôle et anciennes versions sur la page des versions.
